# 🌟 Sky Royale — Android Oyun İstemcisi

Blockman GO Sky Royale modunun özel Android istemcisi.

## 📱 Özellikler
- Koyu tema (dark mode) arayüzü
- Oyun listesi menüsü
- Profil & istatistik ekranı
- Türkçe dil desteği
- Sunucu bağlantı sistemi (yakında gerçek)

## 🚀 APK İndirme
GitHub Actions sekmesine git → En son build → **Artifacts** kısmından APK'yı indir.
